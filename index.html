Gazing with ML

A Machine Learning and Computer Vision event

Brief

The teams will be taught the fundamentals of Machine Learning and Python programming in order to implement a handwriting recognition model. This includes the basics of image processing and using GPU enabled services like Google Colab. They will then be given a different dataset and compete in creating the best performing Machine Learning model with the tools previously taught.

  • Teams of: 2
  • Registration fees: ₹ 200 per team

Round 1 - Hands-on Briefing Session

Teaching the fundamentals of Machine Learning and required linear algebra, to implement a basic object labeling model.

  • Types of Learning - Supervised, Unsupervised, Reinforcement, DL etc.
  • Data Preprocessing - Image convolution, Slicing, Kernels etc.
  • Python Basics - Important libraries, Helper functions.
  • Implementation Walkthrough - Hands-on demo of training a model given a dataset and predicting outputs. (Input - Image, Output - Label)

Round 2 - Competition

Given a labeled dataset containing different pictures, train an ML model to recognize and label the new examples of pictures.

  • Training Dataset - Input: Picture, Output: Labelled Image.
  • Model Objective - Label the given input image.
